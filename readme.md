# VueJs2 Webpack4 Typescript Template
Ready to go VueJs Typescript Template

## usage
* npm start
* npm run build
* npm run build:prod

## included
* vue
* vue-property-decorator
* webpack4 custom build
* typescript
* ts-loader transpileOnly
* scss
* shims for typings
* polyfills via core.js
* tslint
* editorconfig
